# How to make a grid have no focused row


<p>This example demonstrates how to setup a grid so that it initially doesn't have a focused row. The focused row becomes highlighted once you click any.<br><br>If you want not to highlight the focused row at all, set the <strong>GridFocusedRowHighlightHelper.ShowIndicationOnFirstClick</strong> option to <strong>False</strong>.
